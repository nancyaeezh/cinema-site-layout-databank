# cinema-site-layout-databank
该项目围绕电影类站点在页面布局资料库体系与导航路径方面的公开资料进行整理，通过归档栏目命名规则、页面分区逻辑与关键词结构分布，形成系统化研究样本。仓库不包含影视播放或下载内容，仅对公开页面元素进行记录说明。内容重点分析首页与内页之间的层级关系与组织模式，用于页面结构研究。所有资料来源于公开渠道，仅供信息整理与结构分析参考。
